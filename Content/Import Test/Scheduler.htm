Scheduler

Scheduler provides the versatility to customize views and alter data while streamlining actions.

Hide Columns

The Select Columns to Hide feature provides the capability to hide one to multiple columns.

  1. Go to Select Columns to Hide.
  1. Scroll up/down the dropdown list and select all the names of the columns to be hidden.
  1. The top line will tally the number of columns hidden.
  1. Click outside of the dropdown area.
  1. To unhide columns, deselect items that are checked.

 

Sorting Options from the Tool Bar

With the sorting option, one to multiple column titles can be selected to provide a customized sort.

  1. Select Sort in the ribbon.
  1. Under Sort by, select the column for the primary sort.
  1. Additional levels can be added or deleted depending on how granular of sort is required.
  1. Select Add Level to add an additional sort level.
  2. In order to delete a sort level, select the level to unwanted level, then select Delete Level to remove a sort level.
  1. As a default, Ascending order is selected.
  2. Uncheck Ascending order to sort items in a descending order.

 

Filters in Scheduler

Various filter options that are specific to the column selected are available when clicking on the Column Filter Icon.

 

Undo/Red

In order to undo a change that has been made to a specific cell, select the back arrow from the ribbon or enter CTRL Z.

Likewise to redo an entry within a cell, select the forward arrow or enter CTRL Y.

 

Copy/Paste Functions

  1. To copy a row or cell, highlight what is to be copied.
  1. Press CTRL c.
  1. To paste what was copied, select the cell or column.
  1. Press CTRL v.

 

Right Click Options

When right clicking on a cell within the schedule, a window will appear that will provide the capability to Duplicate, Recalculate, add Top Programs, Delete or Clear All Filters.

Duplicate

  1. Duplicate a row by right clicking anywhere within the row.
  2. Select Duplicate.

 

Recalculate

  1. Recalculate rating/impression by right clicking in the row.
  2. Select Recalculate.

 

Top Programs

  1. Get Top Programs by right clicking anywhere within the row.
  2. Select Top Programs.

 

Delete

  1. Delete a row by right clicking anywhere within the row.
  2. Select Delete.

 

Clear All Filters

When right clicking on a cell within the schedule, an option to Clear All Filters will be made available.
