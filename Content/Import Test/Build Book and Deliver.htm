Building a Schedule

Manual Order

  1. Open Scheduler.
  2. Select New and Follow the guided workflow within the platform.

  1. Enter Header Information.
  1. Enter required data. Asterisks indicate mandatory fields.
  1. Note houses custom schedule descriptions (i.e. 2Q Incremental).
  1. Note is visible in the Comments column in Build.
  1. Select Upload, if using .scx file.
  1. Browse folders and select scx file.
  2. Comments pull in by default. Uncheck the option to suppress importing the comments line.
  3. Select Open.

Note: Upload allows .scx files to be imported.  Available header details will pull in from the file.

Flight date can be altered and weeks will adjust accordingly.

  1. Click Apply.
  2. Upon selecting Apply, the SELECT COVERAGE/ACTIVE SYSTEMS window will appear.
  1. Enter Coverage Data.
  1. Select syscode(s)
  1. Choose multiple syscodes by holding Ctrl left click or Shift left click.
  2. ACTIVE SYSTEMS houses all sellable syscodes for the market.

Note:  Integrated resources allow users to identify which syscodes they need by utilizing the ZIP RADIUS or GEO TARGETS tools.

  1. Additionally, the ZIP RADIUS or GEO TARGETS tabs can be selected to identify the criteria for the syscode search.
  1. ZIP RADIUS narrows coverage by entering Zip Code(s) and Zip Radius.
  2. GEO TARGETS is primarily used for political agencies to narrow coverage to GEO TYPE.
  1. Enter STATE.
  2. Select GEO TYPE.
  3. Enter TARGET.
  4. Select GET SYSTEMS.
  1. Click Apply.

 

 

  1. Select Survey Books.
  1. Choose Dataset/Datastream/Survey Period. 
  2. Hold the Ctrl button to select multiple items, then choose ADD, PROJECT or AVERAGE to create a survey book.
  1. Within each category (Market, Geo, Profile and Service) selections must be the same.
  1. Click Apply.

Quickly add programming with easy access to supporting tools all from one location.

  1. Enter 1st line of data, starting with selecting the Network field outlined in red.

Note:  Integration allows programming to be pulled in from ProGo, and to display rate cards from Presto see the Presto section.

  1. Click Add to for single entry add of syscodes, networks or dayparts.
  2. Click Dayparts to add multiple networks and/or dayparts in bulk.
  3. For additional ways to add Programming see section Add Programming.
  1. Save as WIP
  1. From the ribbon, select Save.
  2. WIP ID will display.
  3. Select the syscode(s).
  4. Select SAVE AS WIP.

Note: WIP is available in the Build queue.

WIP can be edited or duplicated from the Build queue.

  1. Save as Order
  1. Select SAVE AS ORDER.
  2. Campaign and Order number will display.
  1. Transmit
  1. If the “header” has not yet been reviewed, selecting Transmit will prompt you to add the Sales Coordinator and review the Conditions of Sale.
  1. Success or Failure of an action taken is reported at the top of window, over the NCC|OPTIX logo.
Things to Note: 
  • An Excel-based platform allows users to customize their view with a simple Click, Drag & Drop of the mouse. (i.e. drag RC Rates next to Historical for a side by side comp.).
  • Ctrl-C to copy / Ctrl-V to paste.
  • Right click any field to reveal a Menu of options.
