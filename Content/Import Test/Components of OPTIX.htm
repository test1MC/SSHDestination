Components of OPTIX

There are five major components to OPTIX.

Dashboard

While in Home, there are two views offered, My Sales and My NCC.

My Sales (future functionality) will provide a customized view of orders based on a particular functional role.

My NCC offers links to key resources and sites in one location, such as NCC internal sites including Support Center, Outlook Web Access, Password Manager, VPN Self Service as well as On the Money, RSS Feed, the Source, etc.


Scheduler

Scheduler is a single platform that is used to create a schedule via a single platform that also seamlessly integrates resources from a wide array of NCC’s tools. While in Scheduler, create headers, select coverage as well as survey books.

 


Build

Build allows for orders to be entered through manual entry, a file import for from a Build Queue.


Book

Book new eOrders in the enhanced OPTIX platform. While in Book, updates and changes can be made to the Header, Scheduler details can be reviewed and the order can be transmitted.


Deliver

Deliver provide the capability of managing orders by checking the status, applying Makegoods, exporting files, viewing orders in the Queue as well as making edits, recalling offers and undoing saved Makegoods.
