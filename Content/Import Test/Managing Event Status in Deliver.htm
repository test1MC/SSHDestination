Managing Event Status in Deliver

Setting the status allows the user to flag events as complete.  Once all events are flagged as complete the “Fire” symbol with update to a check mark to alert the user that Offer is complete.

  1. Locate order in Deliver queue. 

Orders with Offers can be quickly identified by looking for the “fire” symbol in the Events column.

  1. Select Status:
  1. New – Sets action Status to “New” (red flame will appear under events in queue).
  2. In Progress – Sets Action Status to “In Progress” – Events Flag in Deliver queue will display the “Fire” (red flame will remain in queue).
  3. Complete – Set Action to “Complete” - When all actions are flagged complete, the queue will display a green check mark and the flame will go away,     
  1. Add comments.
  2. Select UPDATE EVENTS.
  3. Action Item will be updated with the Status – Comment – User Name & Date.
  4. Select APPLY.
