Queue Views

While in Book, various views can be created, customized and saved.

Creating a View

  1. Select Create View.
  1. Enter View Name.
  1. Select Save.

 

Filtering

  1. In the filter window, enter criteria to be filtered.
  1. Enter

Or

  1. Select flag next to column header name.
  1. Select criteria to filter.
  1. Enter

 

Saving Views

  1. Go to Settings
  1. Select Update.

 

Removing Views

  1. Go to Settings.
  1. Select Remove.

 

Return to Original View

  1. Go to Settings.
  1. Select Original View.

 

Hiding and Adding Columns

  1. Select Settings icon.
  1. Hide columns by removing checkmark.
  1. To make columns visible, select Settings icon and add checkmarks to needed columns.

 

Rearrange Order of Columns

To rearrange the order of the columns, select the column header and drag and drop the column to the desired location.

 

Edit any WIP

  1. Right click on row of desired WIP>
  1. Right click on row of desired WIP.
  1. Select Edit.
  1. Scheduler window will pop up.
  1. Edit Network and/or DP fields.
  1. Select Save from the ribbon.
  1. Select SAVE AS WIP.

Or

  1. Select SAVE AS ORDER.
  1. Enter Comment.
  1. Select SAVE AS ORDER.

OR

  1. Select CANCEL.

 

Duplicate WIP

  1. Right click on row of desired WIP.
  1. Select Duplicate.
  1. Scheduler window will pop up.
  1. If desired, edit Network and/or DP fields.
  1. Select Save from the ribbon.
  1. Select SAVE AS WIP.
  1. The duplicated WIP will appear on the top row and will be assigned a new WIP ID.

OR

  1. SAVE AS ORDER.
  1. Enter Comment.
  2. Select SAVE AS ORDER,
  1. Select CANCEL.

 

eFolders

eFolders provide a means for storing orders electronically on a server.

Accessing eFolders

  1. While in Book, right click on any cell.
  1. Select eFolders.
  1. Left click on file name.
  1. Open pdf file.

 

 

View History

View History tracks activities related to the file.

  1. While in Book, right click on any cell.
  1. Select eFolders.
  1. Select file
  1. Select VIEW HISTORY.

 

Auto Capture Files

Upon transmitting an Order, the Order will be saved to eFolder.
