Additional Features

Add Programming

OPTIX provides a variety of resources to use when adding programming to the Scheduler:

  • Duplicate Existing lines
  • Add New Line / Add To
  • Network
  • Dayparts
  • ProGo
Things to know:
  • The scheduler will launch with a blank buyline/daypart line which can be used to input data.
  • Enter Program Name – or leave blank to pull in Top 5 Programs if available.
  • Spots can be entered in Units/Wk to apply to all visible weeks or in individual weeks

 

Duplicate Existing Buylines

  1. Right Click on a line.
  2. Select Duplicate.
  3. The duplicated line will duplicate in its entirety. Individual fields can be modified as needed.
  4. The duplicated line will appear directly below the selected line.

 

Add to

  1. Select Add To from the ribbon.
  2. Enter line details and select APPLY.
  3. System, Network and DP (daypart) are required fields.
  4. If no program name is entered the Top 5 Programs will populate in the schedule.

 

Network

Network provides the ability so see insert-ability by zone, Genre, Tier and Index.

 

  1. Select Network from ribbon.
  2. Set target audience by selecting from the pulldown menu the Primary Audience and Demo to highlight the appropriate networks for selection.
  1. Add Networks to Schedule
  1. Select Target System – Leave blank to apply to all.
  1. Networks will only be added to systems that have insert-ability.
  1. Select All or Individual Networks.
  1. Ctlr-L click to select multiple items.
  1. Select APPLY.

Dayparts

  1. Select Dayparts from the ribbon.
  2. Select Target System from dropdown.
  1. If target system is not selected, networks will add to all.
  1. Select all or individual networks - Networks will add to the schedule based on insert-ability.
  2. Enter Daypart in text field e.g. M-Sun6p-12m.
  3. Select Add.
  4. Select APPLY to add all selected networks to Scheduler.

 

ProGo

 

  1. Select ProGo from ribbon.
  1. Select HPS (High Profile Series), SPORTS and/or ONE SOURCE tabs.
  1. Sort/filter and search for programming.
  1. Select programming.

  

  1. Select APPLY to add selections to Scheduler.
  1. Comments field will display program start & end dates.
  2. Spots will be added to applicable weeks only.
  3. Right click on line to recalculate or estimated ratings can be manually entered.
