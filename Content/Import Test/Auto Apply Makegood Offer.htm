Auto-Apply Makegood Offer

  1. Locate order in “Deliver” queue. 
  2. Orders with Offers can be quickly identified by looking for the “fire” symbol in the Event column.

 

 

  1. Select the Order in Deliver and right click.

  2. Select View in Events.
  3. Select Action.
  4. View makegood offer(S) will display with Dropped spots and makegood offer (if received from MVPD).
  5. Select View to review offer details.
  6. Offer details will display.

  

  1. Export Report – Offer details will export to a standard Makegood Form.
  2. APPLY OFFER will launch the Scheduler and auto-apply the dropped spots and makegood offer lines.
  3. Scheduler will launch with the changes to dropped spots & new lines.
  1. Dropped Spots –  Changes will highlight in blue. Hover text will display the original spot count and the adjusted count from the Makegood form.
  2. User initiated changes to makegood adjusted lines will highlight in yellow.
  3. Makegood Offer Lines will display as new lines with blue highlighting.

 

  1. If makegood offer cannot be matched – a message will display indicating where the process failed:

 

  1. Once changes have been applied, order can be saved and sent as makegood offer to agency.
  2. Once the Order has been saved with the offer from the Makegood form, APPLY OFFER will be grayed out for the event in the Deliver queue.
