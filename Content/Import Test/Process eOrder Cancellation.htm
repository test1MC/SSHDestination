Process Electronic Cancellation

In Book Queue – Status will display as “Cancellation”.

  1. Right click on order to see available Options.
  2. Select View Details.
  3. Select Syscodes to be cancelled.
  4. Select Cancel from Bulk Actions.
  5. Enter Comment to be sent to MVPT.
  6. Select Recipients.
  1. Select recipients to be added.  Ctrl L to add multiple sections or to “add custom recipient”,
  1. Select ADD SELECTED RECIPIENTS.
  2. The Details page will display the number of recipients.
  3. Select SEND.
  1. A cancellation email will be sent to selected recipients.
  2. The System Status will reflect “Cancellation Pending” on The Spot.
